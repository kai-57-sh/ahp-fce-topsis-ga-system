# 十五个二级指标具体计算公式

## 概述

基于对`modules/evaluator.py`、`modules/scenario_integration.py`、`modules/fce_module.py`等核心代码的深入分析，本文档详细说明了AHP-FCE-TOPSIS-GA评估系统中15个二级指标的具体计算公式。每个指标都包含基础值、修正因子、平台协同效应、任务效率因子、非线性特征和场景调整等计算组件。

## 指标分类

### C1: 态势感知能力 (Situational Awareness Capability)
- C1_1: 侦察探测能力 (Reconnaissance & Detection Capability)
- C1_2: 信息融合能力 (Information Fusion Capability)

### C2: 指挥决策能力 (Command & Decision Capability)
- C2_1: 任务规划能力 (Mission Planning Capability)
- C2_2: 威胁评估能力 (Threat Assessment Capability)
- C2_3: 决策响应能力 (Decision Response Capability)

### C3: 行动打击能力 (Action & Strike Capability)
- C3_1: 火力打击能力 (Fire Strike Capability)
- C3_2: 机动占位能力 (Maneuver & Positioning Capability)
- C3_3: 电子对抗能力 (Electronic Warfare Capability)

### C4: 网络通联能力 (Network & Communication Capability)
- C4_1: 信息传输能力 (Information Transmission Capability)
- C4_2: 网络可靠性 (Network Reliability)

### C1: 态势感知能力

#### C1_1: 侦察探测能力 (Reconnaissance & Detection Capability)
**指标类型**: 效益型 (benefit)
**单位**: capability_score
**基础值**: 50.0 (探测距离km)

**完整计算公式**:
```
C1_1 = Base_50.0 × M_detection × M_stealth × S_platform × S_synergy × E_task × V_nonlinear × A_scenario
```

**计算组件详解**:

1. **基础值**: `Base_50.0`

2. **仿真参数修正因子**:
```python
M_detection = sim_params.get('detection_range_factor', 1.0)
M_stealth = (2.0 - sim_params.get('stealth_factor', 1.0))  # 反向影响
```

3. **平台数量缩放因子**:
```python
# 对数缩放，边际效应递减
S_platform = np.log(total_platforms + 1) / np.log(11)  # 标准化到对数尺度
```

4. **平台协同效应**:
```python
# USV + UUV 组合协同
usv_count = sum(count for platform, count in platform_counts.items() if 'USV' in platform)
uuv_count = sum(count for platform, count in platform_counts.items() if 'UUV' in platform)

if usv_count > 0 and uuv_count > 0:
    S_synergy = 1.15  # 15% 协同提升
    # 额外的平衡混合奖励
    ratio = min(usv_count, uuv_count) / max(usv_count, uuv_count)
    S_synergy *= (1 + 0.1 * ratio)
```

5. **任务效率因子**:
```python
# 基于任务分配覆盖率的效率
E_task = 0.7 + 0.3 * np.sqrt(assigned_platforms / total_platforms)
```

6. **非线性特征**:
```python
# 基于平台组成的微小变化
composition_hash = hash(tuple(sorted(platform_counts.items()))) % 1000
V_nonlinear = 1 + 0.05 * np.sin(composition_hash * 0.1)
```

7. **场景调整因子**:
```python
# 根据作战场景类型调整
if scenario_type == "reconnaissance_surveillance":
    A_scenario = 1.3  # 侦察场景提升30%
```

```python
def _calculate_platform_synergy(platform_counts, indicator_id):
    """计算平台协同效应"""
    synergy_factor = 1.0

    # 检测协同 (USV + UUV 组合)
    if indicator_id in ['C1_1', 'C1_3']:
        usv_count = sum(count for platform, count in platform_counts.items() if 'USV' in platform)
        uuv_count = sum(count for platform, count in platform_counts.items() if 'UUV' in platform)

        if usv_count > 0 and uuv_count > 0:
            synergy_factor *= 1.15  # 15%协同提升
            ratio = min(usv_count, uuv_count) / max(usv_count, uuv_count)
            synergy_factor *= (1 + 0.1 * ratio)  # 额外平衡奖励

    # 指挥协同 (Command平台优化配置)
    elif indicator_id in ['C2_3', 'C4_3']:
        command_count = sum(count for platform, count in platform_counts.items() if 'COMMAND' in platform)
        total_count = sum(platform_counts.values())

        if command_count > 0 and total_count > 0:
            command_ratio = command_count / total_count
            optimal_ratio = 0.1  # 10%指挥平台最优
            synergy_factor *= 1 + 0.2 * np.exp(-((command_ratio - optimal_ratio) ** 2) / 0.01)

    return synergy_factor
```

### 2. 任务效率因子计算
```python
def _calculate_task_efficiency(task_assignments, platform_counts):
    """计算任务分配效率"""
    if not task_assignments:
        return 1.0

    total_platforms = sum(platform_counts.values())
    if total_platforms == 0:
        return 1.0

    assigned_platforms = 0
    for task_data in task_assignments.values():
        if isinstance(task_data, dict) and 'assigned_platforms' in task_data:
            # GA优化器使用的是平台索引列表
            assigned_platforms += len(task_data['assigned_platforms'])

    coverage_ratio = min(1.0, assigned_platforms / total_platforms)
    efficiency = 0.7 + 0.3 * np.sqrt(coverage_ratio)  # 范围：0.7到1.0

    return efficiency
```

## 完整计算架构总结

### 计算公式通用结构
```
最终指标值 = 基础值 × 仿真参数修正 × 平台协同效应 × 任务效率因子 ×
          对数平台缩放 × 非线性特征 × 场景调整因子
```

### 定量指标计算流程
1. **基础值设定**: 根据指标类型设定基础数值
2. **场景调整**: 根据作战场景类型动态调整基础值
3. **仿真参数修正**: 应用检测、协调、武器等参数因子
4. **平台协同效应**: 计算USV+UUV、指挥平台协同加成
5. **任务效率因子**: 基于任务分配覆盖率的效率计算
6. **对数缩放**: 平台数量的对数缩放，边际效应递减
7. **非线性特征**: 基于平台组成的微小变化

### 模糊指标计算流程
1. **专家评价收集**: 收集专家对指标的定性评价
2. **隶属度计算**: 计算各语言变量的隶属度向量
3. **模糊综合评价**: 隶属度向量与模糊评价集的点积
4. **去模糊化**: 获得最终的量化评分

### 关键技术特点
1. **多层级计算**: 从基础值到最终值的层级化处理
2. **场景感知**: 根据作战场景动态调整评估权重
3. **平台协同**: 考虑不同类型平台的协同效应
4. **非线性特征**: 避免评估结果的同质化
5. **模糊评价集成**: 定量与定性评价的无缝结合

这些增强的计算组件确保了AHP-FCE-TOPSIS-GA评估系统能够准确反映无人作战体系在不同作战场景下的真实效能，为指挥决策提供科学可靠的评估依据。
